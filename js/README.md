# E-Hentai Plus

Enhanced continuous reading mode for E-Hentai with advanced features and optimizations.

## Features

- **Continuous Reading Mode** - Scroll through all images seamlessly
- **Parallel Loading** - Fast batch image loading with smart preloading
- **Auto Retry** - Automatic retry on failed images
- **Floating Control Panel** - Quick navigation and settings
- **Dark Theme** - Eye-friendly black background
- **Keyboard Shortcuts** - Efficient navigation controls

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) or similar userscript manager
2. Install the script: [e-hentai Plus.js](https://raw.githubusercontent.com/Leovikii/sm/refs/heads/main/js/e-hentai%20Plus.js)
3. Visit any E-Hentai gallery page

## Usage

The script activates automatically on gallery pages. Use the floating control panel or keyboard shortcuts to navigate.

### Keyboard Shortcuts

- `↑/↓` - Scroll up/down
- `←/→` - Previous/next page
- `Home/End` - First/last page
- `Space` - Toggle auto-scroll

## Performance

- **Parallel loading** - 5 images at once
- **Smart preloading** - Next 10 images ready
- **Optimized for large images** - Handles high-res and WebP animations

## Compatibility

- E-Hentai.org
- ExHentai.org
